Name

cnt_alpha

Synopsis

return_count = cnt_alpha(in_string, sense_flag);
unsigned int  return_count;
const char   *in_string;
int           sense_flag;

Description

Returns the number of (non-) alphabetic characters within a string.

Parameters

Parameters to this function are as follow:

  • in_string is the string in which the count is to be performed.
  • sense_flag specifies the sense of the count to be performed.
    • If sense_flag is non-zero, then this function will count the number of alphabetic characters in in_string.
    • If sense_flag is zero, then this function will count the number of non-alphabetic characters in in_string.

Returns

Returns the number of characters of the desired type within the string.

Notes

An alphabetic character is one for which the macro isalpha returns a non-zero value.
